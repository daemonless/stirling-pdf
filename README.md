## Deployment

### Podman Compose

```yaml
services:
  stirling-pdf:
    image: ghcr.io/daemonless/stirling-pdf:latest
    container_name: stirling-pdf
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
    volumes:
      - "/path/to/containers/stirling-pdf:/config"
    ports:
      - 8080:8080
    restart: unless-stopped
```
